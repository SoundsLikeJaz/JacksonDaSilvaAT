Loja de Eletrônicos

Um projeto simples criado no final da minha disciplina de Desenvolvimento de Serviços Web e Testes com Java onde colocamos em prática nossas hábilidades com consumo de APIs utilizando Spark e, pela primeira vez com Java, fizemos integração com um front-end.
O projeto em questão representa uma loja de eletrônicos que pratica revenda de celulares, computadores e fones de ouvido além de manter um cadastro de seus clientes no sistema (todos podem ser acessados pela navbar acima). Além de cadastrar o cliente, também há uma comunicação com a API Viacep para preencher o endereço do cliente automaticamente após a inclusão de um CEP e há uma relação de um-para-muitos entre cliente e produto, que representa o carrinho de compras do cliente.
